Restrict Manage Attachments

The Restrict Manage Attachment module will restrict a user to add or delete any further attachments as per the access. The default Odoo functionality provided rights to every user to add/delete attachments.

Prerequisites before Installation

No prerequisites needed.

Configurations

No configurations.

Before the installation of this module, you can see in the image every user can add/delete the attachments.

Once you have configured the “Restrict Manage Attachments” from Users> Technical Settings you can pose the restriction for adding/deleting attachments.

You can see after enabling the Restrict Manage Attachments Boolean, user can no more add/delete the attachments.
