Restrict Manage Attachments

Aktiv Software has prepared the Restrict Manage Attachments module with the aim of restricting add and delete attachments based on access right of the user.
  • Any user can add and delete an attachment in ODOO.

Configuration

There is one access right under Technical Settings called Restrict Manage Attachments. With the help of this access right, the user cannot add and delete the attachments.


Restrict Manage Attachments access right

Here, Restrict Manage Attachments access right has no access to add and delete the attachments.
